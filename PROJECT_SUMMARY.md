## 🚀 Быстрый старт

1. **Установка зависимостей**

   ```bash
   npm install
   ```

2. **Настройка .env**

   ```bash
   cp .env.example .env
   # Отредактируйте .env
   ```

3. **Создание БД**

   ```bash
   createdb skyseo
   ```

4. **Миграции**

   ```bash
   npx prisma migrate dev --name init
   ```

5. **Запуск**
   ```bash
   npm run start:dev
   ```

## 📊 API Endpoints

### Публичные

- `POST /auth/register` - Регистрация
- `POST /auth/login` - Вход

### Пользовательские (требуют JWT)

- `GET /users/profile` - Профиль
- `GET /users/balance-history` - История баланса
- `POST /websites` - Добавить сайт
- `GET /websites` - Список сайтов
- `POST /tasks` - Создать задачу
- `GET /tasks/my` - Мои задачи
- `GET /tasks/available` - Получить задачу
- `POST /executions/start` - Начать выполнение
- `PUT /executions/:id/complete` - Завершить
- `GET /statistics/website/:id` - Статистика
- `POST /payments` - Создать платеж

### Админские (требуют роль ADMIN)

- `GET /admin/statistics` - Общая статистика
- `GET /admin/users` - Все пользователи
- `GET /admin/active-users` - Активные сейчас
- `PUT /admin/users/:id/balance` - Изменить баланс
- `GET /admin/tasks` - Все задачи
- `GET /admin/executions` - Все выполнения
- `GET /admin/payments` - Все платежи

## 🔄 Автообновление задач

Для автоматического получения задач каждые 30 минут в клиентском приложении:

```typescript
// В клиентском приложении
setInterval(
	async () => {
		const task = await fetch('http://api/tasks/available', {
			headers: { Authorization: `Bearer ${token}` },
		}).then(r => r.json())

		if (task) {
			// Выполнить задачу
		}
	},
	30 * 60 * 1000,
) // 30 минут
```
