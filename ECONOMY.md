# Экономика баллов SkySEO

## 💰 Приветственный бонус

При регистрации каждый пользователь получает **1000 баллов** бесплатно.
Это примерно **30-40 визитов** на ваш сайт.

## 📊 Как работает система

### Поиск по ключевым словам (SEARCH_KEYWORD)

Программа выполняет следующую последовательность:

1. Поиск в Яндексе или Google по заданному ключевому слову
2. Переход на случайно выбранный сайт из выдачи
3. Посещение нескольких страниц сайта (скроллы, выделение текста, переходы)
4. Снова переход на случайно выбранный сайт из выдачи
5. Снова посещение нескольких страниц
6. Переход на ваш сайт в поисковой выдаче
7. Посещение нескольких страниц вашего сайта

### Переходы по внешним ссылкам (EXTERNAL_LINK)

Программа выполняет:

1. Переход на сайт-донор
2. Поиск вашей ссылки на сайте-доноре
3. Клик по ссылке (если найдена)
4. Посещение нескольких страниц вашего сайта

## 📱 Интеграция с приложением

### Что отправляет приложение

Приложение отправляет только **факт выполнения**, backend сам рассчитывает баллы:

```json
{
	"foundInTop": true, // Найден ли сайт/ссылка
	"position": 5, // Позиция (только для SEARCH_KEYWORD)
	"pagesVisited": 3, // Количество посещенных страниц
	"duration": 120 // Длительность в секундах
}
```

### Что НЕ отправляет приложение

❌ Количество баллов
❌ Тип операции (начисление/списание)
❌ Расчет стоимости

**Все расчеты делает backend!** Это защищает от накрутки.

## 🛡️ Защита от накрутки

### Валидация на backend

1. **Минимальная длительность:** 30 секунд
2. **Максимальная длительность:** 10 минут
3. **Количество страниц:** 2-10
4. **Позиция:** 1-50 (если найдено)
5. **Обязательная позиция:** если сайт найден в топ-50

### Автоматическая блокировка

- Более 100 выполнений за 5 минут
- Более 20 быстрых выполнений (< 10 сек) за час
- Подозрительные паттерны поведения

## 📈 Расчет окупаемости

### Сколько нужно баллов?

**Для продвижения 1 сайта по 10 ключевым словам:**

Если сайт в топ-50:

- 10 ключевых слов × 30 баллов = **300 баллов** за 1 круг
- 10 кругов в день = **3000 баллов/день**
- 30 дней = **90000 баллов/месяц**

Если сайт не в топ-50:

- 10 ключевых слов × 10 баллов = **100 баллов** за 1 круг
- 10 кругов в день = **1000 баллов/день**
- 30 дней = **30000 баллов/месяц**

### Как заработать баллы?

**Запустите программу на заработок:**

- 100 выполнений в день × 5 баллов = **500 баллов/день**
- 30 дней = **15000 баллов/месяц**

**Баланс:**

- Тратите: 30000 баллов/месяц (сайт не в топ)
- Зарабатываете: 15000 баллов/месяц
- Нужно докупить: 15000 баллов ≈ **1250 ₽/месяц**

## ⚠️ Важно

1. **Баллы не возвращаются** если задача выполнена
2. **Минимальный баланс** для создания задачи: 30 баллов
3. **Автоматическое уведомление** при балансе < 100 баллов
4. **Блокировка задач** при балансе < 10 баллов
5. **Честная система** - все выполнения проверяются

## 🔄 API для приложения

### Получить задачу

```http
GET /tasks/available
Authorization: Bearer TOKEN
```

**Response:**

```json
{
	"id": "task-uuid",
	"type": "SEARCH_KEYWORD",
	"keyword": "купить телефон",
	"website": {
		"url": "example.com"
	},
	"geo": "Москва"
}
```

### Начать выполнение

```http
POST /executions/start
Authorization: Bearer TOKEN

{
  "taskId": "task-uuid"
}
```

### Завершить выполнение

```http
PUT /executions/{id}/complete
Authorization: Bearer TOKEN

{
  "foundInTop": true,
  "position": 5,
  "pagesVisited": 3,
  "duration": 120
}
```

**Backend автоматически:**

1. Рассчитает баллы (исполнитель +5, владелец -30)
2. Начислит/спишет баллы
3. Обновит статистику
4. Сохранит историю
